手动摘要：第一使用Hexo博客框架记录一下相关使用的笔记

Welcome to Hexo! This is your very first post. Check documentation for more info. If you get any problems when using Hexo, you can find the answer in troubleshooting or you can ask me on GitHub.

快速启动

新建Hexo项目

$ hexo init 项目名

前提是安装好了Hexo

新建一篇文章

$ hexo new "My New Post"
$ hexo n [模板] 文章名字

More info: Writing

启动服务

$ hexo server
$ hexo s --debug  这样就可以在本地边写边预览啦

More info: Server

生成静态文件

$ hexo generate
$ hexo g

More info: Generating

发布到站点

$ hexo deploy
$ hexo d

More info: Deployment

Hexo配置

主要是修改Hexo根目录下的 _config.yml 文件

详情参考

文章配置

在本件头部编辑
---
title: 文章标题
tags: [多个标签用逗号分隔]
tags: 单个标签
index_img: 目录预览文章图片 /img/example.jpg
banner_img: 文章头部图片 /img/post_banner.jpg
categories: 文章分类
date: 可以自定义发表时间

---
